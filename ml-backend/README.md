# Kornia Warp Engine — ML Backend

FastAPI service that uses **Kornia's Thin Plate Spline (TPS)** transform to warp 2D garment images onto a user's body based on MediaPipe pose landmarks.

## Prerequisites

- **Python 3.11+** (required by Kornia — see [pyproject.toml](https://github.com/Rozuuuuu/kornia/blob/main/pyproject.toml))
- pip

## Quick Setup (Windows)

```bash
# One-click setup
setup.bat
```

## Manual Setup

```bash
# 1. Create virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS/Linux

# 2. Install PyTorch (CPU — for CUDA see https://pytorch.org/get-started/locally/)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu

# 3. Install dependencies (includes kornia, fastapi, uvicorn, opencv, etc.)
pip install -r requirements.txt

# 4. Run the server
uvicorn kornia_warp_engine:app --host 0.0.0.0 --port 8000
# or
python kornia_warp_engine.py
```

## API Endpoints

### `GET /health`
Health check — reports CUDA availability, Kornia version, and torch version.

### `POST /warp`
Warp a flat garment image using TPS transform.

**Request body:**
```json
{
  "garment_image_b64": "<base64-encoded PNG>",
  "landmarks": [
    { "x": 0.35, "y": 0.40 },
    { "x": 0.65, "y": 0.40 },
    { "x": 0.35, "y": 0.75 },
    { "x": 0.65, "y": 0.75 }
  ],
  "anchor_points": [[0.2, 0.2], [0.8, 0.2], [0.2, 0.8], [0.8, 0.8]]
}
```

**Response:**
```json
{
  "warped_image": "<base64-encoded warped PNG>"
}
```

### `GET /docs`
Interactive Swagger UI documentation (auto-generated by FastAPI).

## Architecture

```
Frontend (React/Vite)
    │
    ├─ MediaPipe Holistic → body landmarks (11, 12, 23, 24)
    │
    ├─ Warp Mode Toggle
    │   │
    │   └─ POST /warp ──────► FastAPI Backend (port 8000)
    │                              │
    │                              ├─ Decode base64 garment PNG
    │                              ├─ kornia.geometry.transform.get_tps_transform()
    │                              ├─ kornia.geometry.transform.warp_image_tps()
    │                              └─ Return warped base64 PNG
    │
    └─ Display warped overlay on video feed
```
